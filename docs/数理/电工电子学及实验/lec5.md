本节课主要介绍暂态稳态分析

## 换路定理与初值计算

考虑电路的过渡状态的时候，有两个需要注意的元件，就是**电容**，**电感**

### 换路定理

对于一般情况：

换路前后电容满足：
$$u_C(0_+) = u_C(0_-)$$
换路前后电感满足：
$$i_L(0_+) = i_L(0_-)$$

我们称电路中所求变量（电压或电流）及其$(n-1)$阶导数在$t=0+$的值为电路初值，其中可以等效为独立电源的初值叫做**独立初值**，如电容的电压与电感的电流，可以通过独立初值和电路结构求出的其他初值叫做**非独立初值**

## 暂态瞬态分析

研究一阶电路的全响应：

$$全响应=强制分量（特解、稳态分量）＋自由分量（通解、暂态分量）$$

通用的公式为：
$$f(t) = f(\infty) + \left[ f(0_+) - f(\infty) \right] e^{-\frac{t}{\tau}}$$

也就是我们需要求取的是三个值：

- $f(0_+)$ 初始值：换路后瞬间的值
- $f(\infty)$ 稳态值：稳态后的值
- $\tau$ 时间常数

### RC一阶响应方程

$$u_C(t) = u_C(\infty) + \left[ u_C(0_+) - u_C(\infty) \right] e^{-\frac{t}{R_{eq}C}}$$
其中时间常数为：
$$\tau = R_{eq}C$$

### RL一阶响应方程

$$i_L(t) = i_L(\infty) + \left[ i_L(0_+) - i_L(\infty) \right] e^{-\frac{R_{eq}}{L} t}$$
其中时间常数为：
$$\tau = \frac{L}{R_{eq}}$$
